Exploring K240 disassembles and explores the undocumented game mechanics of the 1994 Commodore Amiga game K240. Read more about this project.

Materials wanted: I'm currently seeking to acquire a copy of the extant original K240 source code disks, the tutorial VHS tape, any surviving Gremlin Interactive design documents or development tools, the French demo version, and magazine review copies, in order to document them at this site. If you have access to any of this data, please reply or add a ticket.
